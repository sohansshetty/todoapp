To-Do List APIs

Create a todo item

Method: POST

URL: /api/tasks/

Request Body: JSON representation of the task to be created

For Example { "title": "Finish project", "description": "Complete the implementation of the To-Do List app", "due_date": "2023-06-30", "status": "OPEN", "tags": ["work", "urgent"] }

Read one todo item

Method: GET

URL: /api/tasks/{task_id}/

Read all todo items

Method: GET

URL: /api/tasks/

Update a todo item

Method: PUT or PATCH

URL: /api/tasks/{task_id}/

Request Body: JSON representation of the task with updated values

For Example { "title": "Working project - update", "description": "Complete the implementation - update", "due_date": "2023-06-30", "status": "OPEN", "tags": ["work", "urgent"] }

Delete a todo item

Method: DELETE

URL: /api/tasks/{task_id}/
